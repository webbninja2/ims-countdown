== Description ==

IMS Countdown timer allows you to display a countdown on your post or page. It help you to stay user in your site.

* Check out the [  [Demo](http://web-xperts.xyz/demo/plugins/ims-countdown/)  )

###⚡️ FEATURES

> * Simple using shortcodes
> * Set the texts for date, time
> * Customize boxes color
> * Customize of colors for text and background
> * Google Fonts for style countdown
> * HTML5: No flash, only jquery
> * Create unlimited styles
> * Redirect to URL option after expiry
> * Hide option after expiry
> * Show link after expiry
> * Timezone option
> * <strong>Fully Responsive</strong> 
> * <strong>Multiple countdown</strong> Timers on the same Post/Page

== Installation ==

1. Go to the Plugins Menu in WordPress
2. Search for **" IMS Countdown "**
3. Click **" Install "**
4. Click **" Active "**
5. Now you can see IMS Countdown menu in sidebar. Where you can add Countdown for your page or post.
